Itch detection AI system team 21

Assistive Robot AI system team

Our goal for this project is to develop a machine learning system that can detect when a person with quadriplegia is experiencing an itch on their face. By using sensors and machine learning algorithms, we hope to create a system that can accurately identify an itch in real-time and alert the person or a caregiver so that appropriate action can be taken to alleviate the discomfort.

Discover More

Abstract

The need for assistive robotics for people with disabilities has become increasingly apparent in recent years. According to the World Health Organization, around 15% of the world's population lives with some form of disability, and many of these individuals struggle with daily tasks that many of us take for granted.

Problem statement​

People like Henry who is quadriplegia are paving the way in assistive robotics. This is the first part of a project to help people like Henry with a range of tasks, starting simply to help him scratch when there is an itch on his face. Our project focus more onto AI face detection which we build a system that helps to detect an itch on clients' faces. After conducting some thorough research we’ve discovered that an itch is caused by a stimulation of nerve endings which could be detected via the use of sensory electrodes, however, this puts convenience and safety into question.​

Solution​​

After brainstorming the solutions of the problem, we were able to reach a solution where we only require a webcam to use video of the clients' faces and an algorithm. Using the video, we capture frames of the facial images to compare the points of the clients' face which compares the facial points of the image. With the comparison of the image, we can insert commands for the clients to use as their needs. This time however, we put the commands as itch detection.​

Achievement and Impact​

We were able to implement above solution and the industrial partner, Avanade was satisfied with our solutions as it can be expanded as multiple solutions and not only the itch detection.
